# sql-to-pandas-example

Convert SQL query to Pandas API

## Installation

```bash
# Create virtualenv
$ virtualenv venv-py37 --python=python3.7

# Active virtualenv
$ source venv-py37/bin/activate

# Install requirements
$ pip3 install -r requirements.txt -I
```

## Get started

```bash
$ python3 process_data.py
```
